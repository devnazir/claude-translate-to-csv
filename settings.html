How it works
  • 1Select an AI provider above and save your API key
  • 2Open any webpage — the Pickslate panel appears automatically
  • 3Click "Start picking" and select text elements on the page
  • 4Choose target languages (up to 14) and set a key prefix
  • 5Hit Translate — results appear in CSV, JSON, or per-language tabs
  • 6Edit results inline, then copy or expand to fullscreen
